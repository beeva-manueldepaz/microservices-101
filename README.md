## Ari

Ari significa hormiga en japones. Por la fabula de Esopo, la cigarra y la hormiga.  

### Prerequisitos

 * Java 8
 * Maven 3.0

#### Construcción

mvn clean package

#### Test

mvn clean test

#### Ejecución

java -jar target/ari-0.1.0.jar

#### Acceptance Test

* cd acceptance-test
* mvn test
